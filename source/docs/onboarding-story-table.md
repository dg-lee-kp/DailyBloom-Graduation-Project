# DailyBloom First-Run Onboarding Story Table

## Goal

첫 접속 사용자에게 DailyBloom의 목적을 짧게 소개하고, 사용자의 생활 목표를 수집한 뒤 초기 습관 3-5개를 추천한다.

AI는 추천안을 바로 저장하지 않는다. 사용자가 `이대로 진행하기`를 선택했을 때만 습관을 생성한다.

## Opening Copy

### Welcome

안녕하세요. DailyBloom에 오신 걸 환영해요.

DailyBloom은 부담 없는 작은 습관을 매일 쌓아가며 생활 리듬을 만들어가는 앱이에요.

작은 씨앗이 매일의 돌봄을 통해 커다란 나무가 되듯, 오늘의 작은 실천이 나만의 성장으로 이어질 수 있게 도와드릴게요.

처음 시작을 위해 몇 가지를 골라 주세요. 선택한 내용을 바탕으로 오늘부터 시작할 수 있는 습관을 추천해드릴게요.

## Step Table

| Step | Question | Input Type | Options | Free Text | Purpose |
| --- | --- | --- | --- | --- | --- |
| 1 | 어떤 방향의 습관을 만들고 싶나요? | Multiple choice | 건강, 공부/성장, 마음관리, 생활정리, 인간관계 | 기타 | 핵심 카테고리 파악 |
| 2 | 하루 중 언제 실천하기 가장 편한가요? | Single choice | 아침, 점심/오후, 저녁, 자기 전, 아직 모르겠어요 | 기타 | 추천 습관의 시간대 조정 |
| 3 | 어느 정도 난이도로 시작하고 싶나요? | Single choice | 아주 가볍게, 적당히, 조금 도전적으로 | 기타 | 습관 크기 조절 |
| 4 | 요즘 가장 방해되는 것은 무엇인가요? | Multiple choice | 피곤함, 시간 부족, 의욕 부족, 잦은 미루기, 스마트폰/콘텐츠 | 기타 | 실패 가능성을 낮추는 추천 |
| 5 | 특별히 원하는 습관이나 피하고 싶은 습관이 있나요? | Free text | 없음 | 직접 입력 | 개인 선호 반영 |

## Option Details

### Step 1: Category

| Option | Recommendation Direction |
| --- | --- |
| 건강 | 물 마시기, 가벼운 산책, 스트레칭, 수면 루틴 |
| 공부/성장 | 책 1페이지, 단어 3개, 강의 5분, 회고 1줄 |
| 마음관리 | 감사한 일 1개, 감정 기록, 호흡 1분, 짧은 명상 |
| 생활정리 | 책상 정리 3분, 빨래 정리, 내일 할 일 1개 적기 |
| 인간관계 | 안부 메시지, 고마움 표현, 가족/친구와 짧은 대화 |
| 기타 | 사용자가 쓴 문장을 AI 추천에 반영 |

### Step 2: Preferred Time

| Option | Schedule Hint |
| --- | --- |
| 아침 | 기상 직후, 등교/출근 전 습관 우선 |
| 점심/오후 | 짧고 장소 제약이 적은 습관 우선 |
| 저녁 | 산책, 정리, 회고, 가벼운 운동 우선 |
| 자기 전 | 수면 준비, 기록, 마음관리 우선 |
| 아직 모르겠어요 | 시간 의존이 낮은 습관 우선 |

### Step 3: Difficulty

| Option | Habit Size |
| --- | --- |
| 아주 가볍게 | 1-3분 안에 끝나는 습관 |
| 적당히 | 5-10분 습관 |
| 조금 도전적으로 | 10-20분 습관 |

### Step 4: Friction

| Option | Recommendation Rule |
| --- | --- |
| 피곤함 | 몸을 크게 쓰지 않는 습관, 자기 전 루틴 추천 |
| 시간 부족 | 1-3분 습관 위주 |
| 의욕 부족 | 완료 기준이 아주 낮은 습관 위주 |
| 잦은 미루기 | 특정 시간/상황에 붙이는 습관 추천 |
| 스마트폰/콘텐츠 | 대체 행동, 짧은 정리, 화면 끄기 루틴 추천 |

## Recommendation Output

AI는 추천 결과를 아래 구조로 만든다.

```json
{
  "summary": "사용자는 건강과 생활 리듬을 만들고 싶어 하며, 저녁 시간에 가벼운 난이도를 선호합니다.",
  "habits": [
    {
      "name": "저녁 5분 산책",
      "description": "집 주변을 짧게 걸으며 하루를 정리하기",
      "schedule": [1, 2, 3, 4, 5]
    },
    {
      "name": "자기 전 물 한 컵",
      "description": "잠들기 전에 물 한 컵 마시기",
      "schedule": [0, 1, 2, 3, 4, 5, 6]
    },
    {
      "name": "내일 할 일 1개 적기",
      "description": "부담 없이 내일 가장 중요한 일 하나만 적기",
      "schedule": [0, 1, 2, 3, 4, 5, 6]
    }
  ],
  "message": "이 습관들로 시작해볼까요?"
}
```

## Confirmation Step

| User Choice | App Behavior |
| --- | --- |
| 이대로 진행하기 | 추천 습관 생성, `is_first_login=false`, 메인 화면 이동 |
| 수정하고 싶어요 | 사용자가 수정 요청 입력, AI가 추천안 재작성 |
| 다시 추천받기 | 같은 선택값으로 다른 추천안 생성 |
| 나중에 할게요 | 습관 생성 없이 메인 화면 이동 여부 확인 |

## Initial Default Recommendation Rules

1. 최초 추천 습관은 3개를 기본으로 한다.
2. 사용자가 여러 카테고리를 고르면 최대 5개까지 추천한다.
3. 모든 습관 이름은 20자 이내를 우선한다.
4. 설명은 한 문장으로 짧게 쓴다.
5. 반복 요일은 기본적으로 매일 `[0,1,2,3,4,5,6]`을 사용하되, 부담이 큰 습관은 평일 `[1,2,3,4,5]`로 제안한다.
6. 사용자가 피하고 싶은 습관을 적으면 추천에서 제외한다.
7. 최종 저장 전에는 반드시 사용자 확인을 받는다.
